Requerimientos de Hardware

Procesador de 1,6 GHz o superior
1 GB de RAM (1,5 GB si se ejecuta en una máquina virtual)
10 GB de espacio disponible en el disco duro
Unidad de disco duro de 5400 rpm
Tarjeta de vídeo compatible con DirectX 9 con una resolución de pantalla de 1024 x 768 o superior

Ultima Actualización(28/07/2020)

Requisitos de Sistema

En Windows 8.1 y Windows Server 2012 R2, se necesita la actualización 2919355 (también disponible a través de Windows Update) para que Visual Studio 2015 se instale correctamente.

Ultima Actualización(28/07/2020)

Requisitos Adicionales

Para el desarrollo de aplicaciones de la Tienda Windows y universales de Windows El desarrollo de Windows 8.1 y Windows Phone 8.1 requiere Windows 8.1 Update o posterior. El desarrollo de Windows Phone 8.0 requiere Windows 8.1 Update (x64) o posterior. Para los emuladores de Windows, Windows 8.1 (x64) Professional Edition o versiones posteriores y un procesador que admita el Cliente Hyper-V y la traducción de direcciones de segundo nivel (SLAT).

Ultima Actualización(28/07/2020)
